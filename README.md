✨ Описание проекта: "Online cafe website":
разработать функциональный и привлекательный веб‑сайт для кафе, который станет цифровым лицом заведения и удобным инструментом для клиентов.

✨ Запуск проекта:
1. Клонируйте репозиторий:
   git clone https://github.com/имя/проект.git

2. Перейдите в папку проекта
    cd проект

3. Откройте файл index.html в браузере.

✨ Использованные технологии:
HTML5, CSS (SASS), JavaScript, БЭМ.
